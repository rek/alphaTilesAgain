# AlphaTiles

*An app generator for literacy games for minority language communities.*

Takes a set of language asset files provided by a language community and generates a mobile app full of literacy games. Rebuilds the original [AlphaTiles Android app](http://alphatilesapps.org/) in Expo + React Native, adding iOS and web support.
