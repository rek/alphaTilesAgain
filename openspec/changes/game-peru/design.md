## Context

`Peru.java` is a 4-choice image-to-word recognition game. The challenge level controls how wrong answers are generated by mutating tiles in the correct word.

## Goals / Non-Goals

**Goals:**
- 3 distinct distractor strategies per CL, each producing 3 distinct wrong choices.
- Unique-choices invariant; reject `┘ä┘ä█ü`-containing candidates.
- Container/Presenter split; i18n-blind presenter.

**Non-Goals:**
- Syllable variant.
- Distractor types beyond same-type and distractor-trio.

### D2. Wrong-Answer Generation per CL

```ts
function buildWrong(level: 1|2|3, parsed: string[], pools): string {
  const mutated = [...parsed];
  if (level === 1) {
    const trio = pools.distractorTrioOf(parsed[0]);
    mutated[0] = randomChoice(trio);
  } else if (level === 2) {
    const i = randomInt(parsed.length);
    const sameType = pools.sameTypeTilesOf(parsed[i]).filter(t => t !== parsed[i]);
    mutated[i] = randomChoice(sameType);
  } else { // level 3
    const i = randomInt(parsed.length);
    const trio = pools.distractorTrioOf(parsed[i]);
    mutated[i] = randomChoice(trio);
  }
  return mutated.join('');
}
```

### D3. Uniqueness + Forbidden-Substring

```ts
const FORBIDDEN = '┘ä┘ä█ü';
function buildAllChoices(correct: string, level, parsed, pools): string[] {
  const choices = new Set<string>([correct]);
  let guard = 0;
  while (choices.size < 4 && guard++ < 200) {
    const candidate = buildWrong(level, parsed, pools);
    if (candidate.includes(FORBIDDEN)) continue;
    if (candidate === correct) continue;
    choices.add(candidate);
  }
  return shuffle([...choices]);
}
```

If `guard` is exceeded (degenerate language pack), choose another `refWord`.

### D4. Image Tap

Tapping the word image MUST replay the active word clip (Java `playActiveWordClip`).

### D5. Container / Presenter Split

**`<PeruContainer>`** ŌĆö owns:
- `useGameShell()`, `useLangAssets()`, `useAudio()`, `useProgress()`.
- State: `prompt`, `choices`, `correct`, `disabled`, `wrongPicks`.
- Handlers: `onChoice`, `onPlayAgain`, `onRepeat` (also fires on image tap).

**`<PeruScreen>`** ŌĆö pure props ŌåÆ JSX:
- `wordImage`, `choices: { text: string; grayed: boolean }[]`, `onChoice(text)`, `onRepeat`.
- No hooks; no i18n.
